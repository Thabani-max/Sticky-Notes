# Sticky Notes Project
---

## Description

This is a sticky notes django app. The user of the app is able to create a new post, with a title and post contents, they can also view
the list of posts saved and, view the details of each post. The app also allows users to Update a posts' content and/or delete a post.

### Implementing the Project

To implement the project, you should first replicate the project environment used to create the project by creating and activating a vertual environent for the project and installing the tools inside the 'requirements.txt' file inside the project's main directory.

After this has been done, in your terminal, navigate to the main directory of the project ('sticky_notes) and enter the command,
'''
python manage.py runserver
'''
and then click on the link generated (**http://localhost:8000/.**) to run the app. Here you can Add a post, view the list of existing posts, view a particular posts' details, and update or delete an existing post in your sticky notes app.
